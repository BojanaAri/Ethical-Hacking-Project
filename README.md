# Ethical-Hacking-Project
OWASP Top 10 in a PHP-based development framework (Laravel Web App)
Implementation of a web application in a popular web framework based on the PHP programming language, which would intentionally contain OWASP Top 10 vulnerabilities. Review of the security traps that programmers can fall into, but also the built-in capabilities that the framework itself offers to protect against these security threats (SQL injection mitigation, CSRF tokens...). Evaluation of third-party solutions for scanning vulnerabilities in code and their effectiveness
